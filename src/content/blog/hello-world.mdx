# Hello World!

Welcome to the **Modern MDX Blog Template**. This is a sample blog post to demonstrate the styling and features of the template.

## Features

This blog template comes with a lot of features out of the box:

- **MDX Support**: Write your content in MDX, which allows you to use React components in your markdown.
- **Syntax Highlighting**: Code blocks are styled with syntax highlighting.
- **Dark Mode**: Switch between light and dark mode with ease.
- **Responsive Design**: Looks great on all devices.
- **SEO Optimized**: Built with SEO in mind.
